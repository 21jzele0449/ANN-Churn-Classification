# Churn Prediction App (ANN + Streamlit)

##  Overview

This project is a Machine Learning web application that predicts whether a customer is likely to churn or not. It is built using an Artificial Neural Network (ANN) and deployed with Streamlit for an interactive user interface.

The model is trained on the **Churn Modelling dataset** and helps businesses identify customers who may leave, enabling better retention strategies.

##  Features

* Predict customer churn in real-time
* Interactive web interface using Streamlit
* Data preprocessing with encoding and scaling
* Deep learning model built using TensorFlow/Keras
* Displays prediction probability


## Technologies Used
* Python
* TensorFlow / Keras
* Scikit-learn
* Pandas & NumPy
* Streamlit


## Model Details
* Model Type: Artificial Neural Network (ANN)
* Input Features: Customer demographics & account information
* Output: Binary classification (Churn / Not Churn)
* Evaluation Metrics: Accuracy, Precision, Recall
